Neural Concept Recognizer

NCR is a concept recognizer for annotating unstructured text with concepts from an ontology. In its core, NCR uses a deep neural network trained to classify input phrases with concepts in a given ontology, and is capable of generalizing to synonyms not explicitly available. Currently NCR is trained to recognize concepts from Human Phenotype Ontology (HPO).

If interested in using the RESTful API, please check the documentation page.
